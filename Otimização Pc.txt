							Otimização Do Pc

1. Na Bios,Escolha a Menor Opção Para Memoria Para Hardware.

2. Troque o Navegador Por Navegador Opera.
	-Instale Uma Extensão de tradutor e baixar videos do youtube. no Opera
	
	-Desative o navegador padrão 
		-painel de controle - desinstalar programa - microsoft edge
		
		-tecla windows - Painel de Controle - programas e recursos - ativar ou desativar recursos - Marcar Internet Explorer
		
		-tecla windows+R - "digite Regedit" - clique em "Editar" - Pesquise Pelo Nome Do Programa
			-Edge, Microsoft Edge, Microsoft\edge, Microsoft-edge
	
3. Não Instalar Anti-Virus 
	-O Proprio Sistema Ja Trabalha Com Windows Defender
	
4. Desativar Serviços Desnecessarios Método 1.
	- Tecla windows - Painel de Controle - programas e recursos - ativar ou desativar recursos
		- Marcar Internet Explorer
		- Serviços Xps
		- Ver Pdf Dedicado a Isso.

5. Desativar Serviços Desnecessarios Método 2.
	- Tecla Windows - digite "Services.msc"
		
		- Desativar Spooler de Impressão - Se Não Houver Impressora
		
		- Desativar Logon de rede (Netlogon) - Autentica a sua conta de usuário com um controlador de domínio. Isso é comum em empresas, mas o seu computador pessoal 			certamente não faz parte de um domínio, deve desativar.
		
		- Serviço de Suporte a Bluetooth - Se não ter Bluetooth no computador.
		
		- Desativar Logon secundário - Permite executar tarefas administrativas como uma conta padrão utilizando a opção “Executar como. Se você possui somente uma conta 		   em seu PC, você não precisa desse recurso e pode desativá-lo.

		- Desativar Cartão Inteligente
	
		- Desativar Configuração da Área de Trabalho Remota

		- Desative Fax
	
		- Desativar Propagação de certificado 
		Se você não estiver usando SmartCards, desative-o.		
		
6. Limpezas
	- Limpeza de Disco
	- Limpar Pontos de Restauração Antigos ou Desnecessários
		- Propriedades do Sistema - Proteção do Sistema - Configurar - Excluir.
	- Limpar Cache Navegador
 		
7. Desabilitar Programas Que Não Necessitam Iniciar ao Ligar Máquina.
	- Gerenciador de Tarefas - Inicialização
	
8. Desabilitar Assistência Remoto
	- Propriedades do Sistema - Proteção do Sistema - Remoto.
	
9. Desabilitar Efeitos Visuais
	- Propriedades do Sistema - Proteção do Sistema - Avançado - Desempenho - Escolha As opções
		-Deixar Marcado as Opções
			- Abrir caixas de combinação
			- Mostrar Conteúdo Da Janela ao Arrastar
			- Mostrar Miniaturas ao invés de ícones
			- Rolar Caixa De Lsitagem Suavemente
			- Usar Fontes de Telas Com Cantos arredondados
